my music lab

hi!!! this project is so all over the place right now it's actually kind of embarassing

This is [going to be?] a series of experiments of music in code. I've made a few projects before in tune with this idea, which are now available here:

  • astronomy487.github.io/musiclab/progression-o-tron A tool to extend chord progressions using the power of graph theory and a sample bank of chord progressions. I just used a few songs from the real book because sheet music was readily available online
  • astronomy487.github.io/musiclab/voiceleader A tool to do 4-part voice leading, following all the silly rules they teach you about in ap music theory

These projects were accomplished using some kinda janky code. I've recreated the functionality needed for these tools in my new little javascript library for interfacing with chord notation and stuff:

  • chordanalysis.js the all-important library
  • vexflowwrite.js a mini mini library to work with vexflow, a really good library for making notation. i made this interface so i can easily write simple quarter note chords

But I haven't actually recreated the PROGERSSION-O-TRON and VoiceLeader yet using this library. That's still in the work.

You can look at my silly experiments in the meantime:

  • chordidentifier gimme a chord name, i give you the notes it's made of
  • vexflowwrite just uses the mini mini library i created to interface with vexflow. a little experiment to throw test cases at my work
  • voicer a first draft of a way to create voice leading. actually works pretty nice, but i need to make it work more nicely with the above vex flow write mini mini library

There's a lot to do! But there's plenty of time!
